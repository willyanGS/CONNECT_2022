Lista de exercícios 3 – Javascript
Todos os exercícios deverão ser respondidos com pares de arquivos html/Javascript. 
Todos os exercícios deverão ter um título dentro da <head>.
O texto “Exercício x” deverá estar dentro de tags <h1> 
O enunciado do exercício deverá estar dentro de uma <p>

1 – Crie um documento que contenha um botão com o texto “meu botão”.
Crie uma função que altere o texto do botão para “meu segundo botão”.

2 – Crie um parágrafo com o texto: “Execute a função para exibir seu nome”
Crie uma função que receba um nome por prompt;
Modifique o texto do parágrafo para: “Tenha um bom dia, sr (a) nome”

3 – Crie um input do tipo texto.
Crie uma função que envie um alert com o texto escrito no input.

4 – Crie um input do tipo texto que possua o placeholder “digite seu cpf”.
Crie uma função que verifique se o cpf informado possui exatamente 11 digitos.
Crie alertas para quando o cpf estiver errado.
Escreva o tipo do erro em um <p> abaixo da caixa de texto
Se estiver correto, crie uma caixa de confirmação.
Se clicar em OK, Exiba o texto “CPF Válido”
Se clicar em cancelar, exiba o texto: “operação cancelada”
